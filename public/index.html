td-registry

Thomas Digital private shadcn registry. Channel-based versioning per ADR-02.

Active channels:

  • @td-v1 — active
  • @td-v2 — preview

Manifest: /r/registry.json

Client setup (clients pin one channel):

// components.json
{
  "registries": {
    "@td-v1": "https://registry.thomasdigital.com/r/v1/{name}.json"
  }
}

Then: pnpm dlx shadcn@latest add @td-v1/<name>
